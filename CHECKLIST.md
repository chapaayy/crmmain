# MVP Acceptance Checklist

- [ ] Auth работает: login, refresh, logout, `/auth/me`.
- [ ] Роли работают: RBAC скрывает UI-разделы и защищает API endpoints.
- [ ] Товары создаются и редактируются.
- [ ] Клиенты создаются и редактируются.
- [ ] Заказ создается.
- [ ] Сумма заказа считается backend: subtotal, discount, tax, total, payment status.
- [ ] Резерв склада работает.
- [ ] Нельзя уйти в отрицательные остатки.
- [ ] Счет PDF создается и скачивается через защищенный API.
- [ ] Dashboard открывается и показывает пустые/demo states без ошибок.
